Class TBase

Description
Description | Vars (details) | Methods (details)

Base class from which other Thrift structs extend. This is so that we can

cut back on the size of the generated code which is turning out to have a nontrivial cost just to load thanks to the wondrously abysmal implementation of PHP. Note that code is intentionally duplicated in here to avoid making function calls for every field or member of a container..
